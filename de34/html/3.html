Flask

FlaskとはPythonでWebアプリケーションを作るための軽量なフレームワークであり、
初心者や学習用として向いている。

基本構成

  • ルーディング（URLと処理の対応）
  • テンプレート（HTML表示）
  • リクエスト処理（GET/POST）
  • 静的ファイル（CSS・画像など）

使用例

    from flask import Flask, render_template, request, session, redirect,flash, url_for
from csv import reader

app = Flask(__name__)

# 編集ゾーンーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
##ここでルートのURLが指定された時に実行される
@app.route('/')
def index():
    #index.htmlがレンダリングされる
    with open('./mmcsv.csv', 'r', encoding="utf-8") as csv_file:
        csv_reader = reader(csv_file)
        list_mm = list(csv_reader)

    for data in list_mm:
        data[3] = "https://www.google.com/maps?q=" + data[3].replace("(","").replace(")","").replace(" ","")

    print(list_mm)
    return render_template("index.html",mmlist=list_mm)

# ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
if __name__ == '__main__':
    app.run()

※__name__とはPythonに存在する特別な変数で、
このファイルがどこから実行されているかをFlaskに伝えるものである。

フロントエンドとバックエンド

アプリケーションなどを作成するにあたってフロントエンドとバックエンドが存在する。

フロントエンド

ユーザーの目に見える情報

  • Webページのデザイン（ボタン・文字・画像）
  • CSS（見た目・デザイン）
  • JavaScript（動き・インタラクション）

バックエンド

ユーザーから見えない裏側の処理

  • ログイン情報のチェック
  • サーバー処理
  • データベースへの保存

※Flaskではバックエンドをしている。

データベース

データベースとは、バックエンドがデータを保存・取得するために使う外部の記録場所のことである。
